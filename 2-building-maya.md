# Software Setup

Maya’s brain is a microcontroller that is designed by Sparkfun called a Pro Micro. It is essentially a very tiny portable computer capable of sensing inputs from the outside world (such as how bright it is from light sensors), then uses this data to decide how to control various outputs (such as moving its motors to go backward and forward). This board is capable of taking the world of electronics and relating it to the physical world in a real and tangible way (which is where the term “physical computing” comes from).

In order to play with Maya, we need to make sure Maya’s brain is working. To get it up and running . You’ll need access to the internet using a computer. You need to set-up a Codebender  or an Arduino IDE software. You will also use this software to configure Maya’s brain to do exactly how you want Maya to behave (This is also called coding, which you will learn later!)

# Check Peripherals

Next, let’s make sure Maya’s electronic peripherals are functioning properly. Maya has a separate board where you can hook-up her brain along with various modules. Hook them up and make sure all of the components are working properly by following the video below. Aside from hooking it up, make sure you have the Maya libraries properly installed before you run the demo programs/code. You can think of programs/code as a special language you write that Maya’s brain can interpret how you want her to behave. A software library is pre-written code that you can use to make it easier for you to write your own code, so you don’t have to write everything from scratch yourself.
